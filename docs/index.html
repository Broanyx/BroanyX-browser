Everything you need to stay private

🧅

Tor Network Routing

All traffic is automatically routed through the Tor anonymity network. Your real IP is always hidden.

🚫

Network-Level Ad Blocking

Blocks ads, trackers, and malware domains using EasyList + EasyPrivacy rules before they reach the page.

🔒

WebRTC & Geolocation Disabled

Prevents WebRTC IP leaks and disables geolocation APIs that could expose your location.

🕵️

User-Agent Spoofing

Randomizes your browser fingerprint so websites can't track you across sessions.

🔄

Auto-Updates

The browser silently checks for updates on launch and notifies you in-app when a new version is ready.

🌐

Dark Web Support

Browse .onion sites natively through the built-in Tor integration. No extra config needed.

🔄 Always Up to Date

BroanyX checks for updates automatically every time you open it.
When a new version is released, you'll see a notification banner inside the browser with a direct download link — no manual checking needed.
